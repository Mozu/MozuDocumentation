Name Nullable Description Data Type List
accessToken No The token that stores an encrypted list of the application's configured behaviors and authenticates the application. string No
accessTokenExpiration No Date and time the access token expires. After the access token expires, refresh the authentication ticket using the refresh koken. DateTime No
refreshToken No The token that refreshes the application's authentication ticket. string No
refreshTokenExpiration No Date and time the refresh token expires. After the refresh token expires, generate a new authentication ticket. DateTime No
